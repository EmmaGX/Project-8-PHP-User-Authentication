# Project 8 PHP User Authentication
 Unit 8 Project with Treehouse Learning

1. Allow new users to register for your application. 

   
2. Store the user's password as a hash.


3. Add the ability for a user to login to your application.


4. Add the ability to log out of your application.


5. Allow users to update their password.


6. Assign all new tasks to the logged in user.


7.Only return tasks belonging to the logged in user
ONLY the home, registration, and login pages should be accessible for unauthenticated visitors. All other pages should require authentication.
